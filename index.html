This is a project I worked on with my group for Hack the North 2018. It's a web application that utilizes TensorFlow.js and the PoseNet machine learning model to estimate a user's form and provide feedback for improvement. The model itself provides the computer vision capability to estimate form from pictures, and provides data about the user's limbs and joints. The web application portion was developed using Node.js + Express.
